# 🍳 CookMind AI

CookMind AI is a modern AI-powered food assistant that helps users discover recipes, cooking tips, nutrition advice, and personalized meal ideas through an interactive chat experience.

---

## 🚀 Live Demo
👉  https://cookmind-pd6g027l9-harshitas-projects-b0b83ea6.vercel.app

---

## ✨ Features

- 🤖 AI-powered cooking assistant
- 🍜 Smart recipe suggestions
- 🥗 Healthy diet & nutrition guidance
- 💬 Chat-based AI interaction
- 🎤 Voice input support
- ⚡ Fast responses using Gemini API

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- CSS3 (Modern UI / Glassmorphism)
- Gemini API
